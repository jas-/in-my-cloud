Welcome to Cerberus!


The first PHP development framework that implements public key encryption for authentication, management of database objects, AES encryption for session objects and more, such as...

  1. Multiple authentication sources
  2. Tiered access level support
  3. User & Group permission support
  4. User public key encryption for all form data
  5. AES encryption support for all database objects
  6. Easy setup and configuration
  7. Extensible as framework for future applications

This project is in the alpha stages of development, but if your interested in assisting with development, testing or just general usage you can sign up here....
